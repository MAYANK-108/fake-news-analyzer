# 🔍 Fake News Analyzer

An AI-powered fake news detection app built with Python and Streamlit, using Groq AI (Llama 3.1) to analyze and verify news authenticity.

## 🌐 Live Demo
👉 **[fake-news-analyzer-ai.streamlit.app](https://fake-news-analyzer-ai.streamlit.app)**

---

## ✨ Features

- 🔍 Detects if news is **REAL, FAKE, MISLEADING or UNVERIFIED**
- 📊 Confidence level percentage for each analysis
- 🚩 Red flags detection — suspicious claims, missing sources, emotional language
- ✅ Suggestions on where to verify the news
- ⚡ Powered by **Groq AI (Llama 3.1)**

## 🛠️ Tech Stack

- **Python**
- **Streamlit** — for the web UI
- **Groq API** — for AI-powered news analysis
- **python-dotenv** — for environment variable management

---

## 🚀 Getting Started

### 1. Clone the repo
```bash
git clone https://github.com/MAYANK-108/fake-news-analyzer.git
cd fake-news-analyzer
```

### 2. Install dependencies
```bash
pip install -r requirements.txt
```

### 3. Set up your API key
Get a free API key from [console.groq.com](https://console.groq.com)
Create a `.env` file:
```
GROQ_API_KEY=your_api_key_here

```
### 4. Run the app
```bash
streamlit run Newz.py
```
